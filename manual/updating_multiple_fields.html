Updating multiple fields at a time

When viewing an entry you have write privileges on, the Edit button in the action menu will be enabled. Clicking on it will show details about the entry in the edit form. Make any desired changes and click the “Submit” button to save them. Click “Cancel” to cancel the edit process and return to the detail view of the entry.
