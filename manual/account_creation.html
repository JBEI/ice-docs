Account Creation

Download Video



For ICE installations that allow users to create accounts, there is a Create Account button on the login page. Click on it to navigate to the Account Creation page. Enter your name, email address, institution affiliation and brief information about you. Click Submit to create the account and a temporary password will be emailed to you.


If the Create Account button is not available then it implies that the registry instance does not permit user registrations and might be integrated with another user directory system. In that case contact the administrator.

If you get an error message when creating the account, then it is likely that the email address you used already has an account associated with it. In that case, use the Forgot Password feature to reset your password.
