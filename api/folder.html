Folder Resource

Title Get Folder
Retrieves details of a folder
URL /rest/folders/:id
Method GET
URL Params Required:
  • id=[integer]
    Unique identifier for folder
    example: id=12

Optional:
None
Data Params
Header Params X-ICE-Authentication-SessionId
Unique session id obtained after most recent successful authentication.

OR

X-ICE-API-Token-Client AND X-ICE-API-Token
API Client name and token available from API Keys
Success Response Code: 200 OK
Example content: Show Hide 
{
    "folderName": "Hillson et al. 2012",
    "count": 0,
    "propagatePermission": false,
    "type": "PUBLIC",
    "publicReadAccess": false,
    "canEdit": false,
    "id": 2,
    "creationTime": 0,
    "entries": []
}
Error Response Code: 401 UNAUTHORIZED
If session identifier is invalid

Code: 403 FORBIDDEN
If url parameter is set but API token is invalid

Code: 404 NOT_FOUND
If folder resource cannot be located using specified identifier

Code: 500 INTERNAL_SERVER_ERROR
On unknown server error
Sample Call 
curl -X GET
     -H "Content-Type: application/json"
     -H "X-ICE-API-Token-Client: ‹api-token-client›"
     -H "X-ICE-API-Token: ‹api-token›"
     -H "Cache-Control: no-cache"
     -H "https://public-registry.jbei.org/rest/folders/2"
Notes The unique identifier for the folder can be obtained from the list of folders retrieved from the call to /rest/collections/{type}/folders
Title Get Folder Entries
Retrieves entries of a folder
URL /rest/folders/:id/entries?offset=:offset&limit=:limit
Method GET
URL Params Required:
  • id=[integer]
    Unique identifier for folder
    example: id=12

Optional:
  • offset=[integer] default=0
    Paging parameter start.
    example: offset=15
  • limit=[integer] default=15
    Maximum number of records to return
    example limit=30
Data Params
Header Params X-ICE-Authentication-SessionId
Unique session id obtained after most recent successful authentication.

OR

X-ICE-API-Token-Client AND X-ICE-API-Token
API Client name and token available from API Keys
Success Response Code: 200 OK
