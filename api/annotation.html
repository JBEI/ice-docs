Title Get Annotations Admin
Retrieves list of available sequence annotations
URL /rest/annotations?offset=:offset&limit=:limit&sort=:sort&asc=:asc
Method GET
URL Params Required:
None

Optional:
  • offset=[integer] default=0
    Paging parameter start.
    example: offset=15
  • limit=[integer] default=15
    Maximum number of records to return
    example limit=30
  • sort=[string] default='created'
    Sort field
  • asc=[boolean] default=true
    Sort order example: asc=false
Data Params
Header Params X-ICE-Authentication-SessionId
Unique session id obtained after most recent successful authentication.

OR

X-ICE-API-Token-Client AND X-ICE-API-Token
API Client name and token available from API Keys
Success Response Code: 200

Title Rebuild Annotations Admin
Rebuilds the annotations database used in the auto-annotations feature.
URL /rest/parts/:id
Method PUT
URL Params Required:
None

Optional:
None
Data Params
Header Params X-ICE-Authentication-SessionId
Unique session id obtained after most recent successful authentication.

OR

X-ICE-API-Token-Client AND X-ICE-API-Token
API Client name and token available from API Keys
Success Response Code: 200 OK
Error Response Code: 401 UNAUTHORIZED
If the account of the user making the request does not have administrative privileges

Code: 500 INTERNAL_SERVER_ERROR
On unknown server error
Sample Call 
curl -X PUT
     -H "Content-Type: application/json"
     -H "X-ICE-API-Token-Client: ‹api-token-client›"
     -H "X-ICE-API-Token: ‹api-token›"
     -H "Cache-Control: no-cache"
