Web Resource

Enables interaction with web of registries
Title Get Web Entries
Get entries that are available on web of registries
URL /rest/web/entries?download=:download
Method GET
URL Params Required:
None

Optional:
  • download=[Boolean]
    Whether to package the entries and make it available for download.
    example: download=true
Data Params
Header Params X-ICE-Authentication-SessionId
Unique session id obtained after most recent successful authentication.

OR

X-ICE-API-Token-Client AND X-ICE-API-Token
API Client name and token available from API Keys
Success Response Code: 200 OK
Example content: Show Hide 
{
    key: "fileName",
    value: "zip-2807431090184812821.zip",
}
Error Response Code: 401 UNAUTHORIZED
If session identifier is invalid

Code: 403 FORBIDDEN
If url parameter is set but API token is invalid

Code: 404 NOT_FOUND
If resource cannot be located using specified identifier

Code: 500 INTERNAL_SERVER_ERROR
On unknown server error
Sample Call 
curl -X GET
     -H "Content-Type: application/json"
     -H "X-ICE-API-Token-Client: ‹api-token-client›"
     -H "X-ICE-API-Token: ‹api-token›"
     -H "Cache-Control: no-cache"
     -H "https://public-registry.jbei.org/rest/web/entries?download=true"
Notes To download the file, make a GET to rest/file/tmp/{result.value} where result is the success response object
