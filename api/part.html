Title Create Part
Create a new part in ICE
URL /rest/parts?source=:sourceId
Method POST
URL Params Required:
None

Optional:
  • source=[string]
    One of the unique identifier values for an existing part (part number, uuid or id). A copy of that part is created if set
    example: source=12 or source=Jbx_01234 or source=067e6162-3b6f-4ae2-a171-2470b63dff00
Data Params Note: If the source query param is set, this will be ignored

{
    type: 'STRAIN' | 'PLASMID' | 'PART' | 'ARABIDOPSIS',
    name: String,
    alias: String,
    keywords: String,
    status: 'Complete' | 'In Progress' | 'Planned' | 'Abandoned',
    shortDescription: String,
    longDescription: String,
    references: String,
    bioSafetyLevel: Integer,
    intellectualProperty: String,
    links: [String],
    principalInvestigator: String,
    principalInvestigatorEmail: String,
    selectionMarkers: [String],
    fundingSource: String,
    parameters: [{name: String, value: String}],
    strainData: { // if type='STRAIN'
        host: String,
        genotypePhenotype: String
    },
    plasmidData: { // if type='PLASMID'
        backbone: String,
        originOfReplication: String,
        promoters: String,
        circular: Boolean,
        replicatesIn: String
    },
    arabidopsisSeedData: { // if type='ARABIDOPSIS'
        homozygosity: String,
        ecotype: String
        harvestDate: String
        seedParents: String
        generation: UNKNOWN | F1 | F2 | F3 | M0 | M1 | M2 | T0 | T1 | T2 | T3 | T4 | T5,
        plantType: EMS | OVER_EXPRESSION | RNAI | REPORTER | T_DNA | OTHER | NULL,
        sentToAbrc: Boolean
    }
}
Header Params X-ICE-Authentication-SessionId
Unique session id obtained after most recent successful authentication.

OR

X-ICE-API-Token-Client AND X-ICE-API-Token
API Client name and token available from API Keys

}

If session identifier is invalid

Code: 500 INTERNAL_SERVER_ERROR
On unknown server error

Title Delete Part Experiment
Deletes a part experiment link for the specified part
URL /rest/parts/:id/experiments/experimentId
Method POST
URL Params Required:
  • id=[string]
    One of the unique identifier values for a part (part number, uuid or id)
    example: id=12 or id=Jbx_01234 or id=067e6162-3b6f-4ae2-a171-2470b63dff00

  • experimentId=[long]
    Unique identifier for experiment link. Can be obtained from list of experiments or data returned after creation
    example: experimentId=13

Optional:
None
Data Params
Header Params X-ICE-Authentication-SessionId
Unique session id obtained after most recent successful authentication.

OR

X-ICE-API-Token-Client AND X-ICE-API-Token
API Client name and token available from API Keys
Success Response Code: 200 OK
Error Response Code: 400 BAD_REQUEST
If the partId parameter does not correspond to an entry

Code: 401 UNAUTHORIZED
If session identifier is invalid

Code: 403 FORBIDDEN
If requesting user's account does not have write privileges on the entry or has read privileges but did not create the experiment link

Code: 500 INTERNAL_SERVER_ERROR
On unknown server error
Sample Call 
curl -X DELETE
     -H "Content-Type: application/json"
     -H "X-ICE-API-Token-Client: ‹api-token-client›"
     -H "X-ICE-API-Token: ‹api-token›"
     -H "https://public-registry.jbei.org/rest/parts/226/experiments/15"
Notes Users with read privileges on parts are permitted to create and delete links. In order to delete a link on an part which the user has read access on, they must be the creator of the link.
